Golf Clash

Here is a look at the very first project I built with JavaScript. The inspiration behind this game came from my passion for sports. The technologies used to build this game include JavaScript (canvas), HTML, and CSS. Give it a try!

Data Benefits

The first project I built that included a backend JavaScript Framework, Express.js with authentication required. I was inspired to create this application so that users can see first-hand what data they are providing to companies, and allow them the opportunity to use that data to their benefit.

Inventory Management Assistant

A partner and I built this full-stack application designed to make it easier for business owners to manage their inventory and track their personnel. The technologies used include MongoDB, Express.js, Node.js, and React.js. This taught me a lot about git and merging code, and also introduced me to my favorite JavaScript library, React.JS.

DiMaggio Realty LLC

This static website was created for the property management group, DiMaggio Realty LLC. It's an informative site that is meant to inform potential renters about the properties that they have available. Built with HTML, CSS, and Bootstrap - it is a continuous work in progress.

Rain or Shine

Another web-application built using React.js along with an open source weather API called Open Weather Map. It's functionality is straight forward, you simply input the city and two-letter country code of the location that you wish to access the weather. Feel free to give it a try!

Darts Scoreboard

Another React.js application inspired by one of my favorite hobbies, darts. If you are familiar with darts, then you may have come across a few situations where you have to get the pen and paper in order to keep score. This web-app helps ease that pain, and allows you track points as well. Give it a shot!
